QU'EST-CE QUE LE HTML?

L'abréviation d’Hypertext Markup Language (langage de balisage hypertexte)

HTML

Le HTML est un langage de balisage. Il s'agit du langage le plus important sur le Web. Il définit le contenu d’une page Web.

Comme les langues que nous parlons et écrivons, le HTML suit aussi des « règles de grammaire ». Les balises HTML prennent des chevrons vers la gauche < et la droite > et une barre oblique /. Familiarisez-vous avec ces touches!

Qu'est-ce que du code « figé » ou « codé en dur »?

Du texte écrit explicitement dans le HTML est considéré comme « figé » ou « codé en dur ». Cela signifie qu'il ne peut être modifié que par quelqu'un qui a accès aux fichiers du site et qui a des connaissances en programmation.

HTML


<h1>Bottes poilues</h1>
<h3>139,99 $</h3>
<p>Ces bottes vous garderont au chaud tout l'hiver. Elles sont super!</p>

Quelle est l'utilité de Liquid?

Plutôt que d'écrire explicitement ce que nous voulons, nous pouvons faire référence à un objet Liquid.

Les objets Liquid affichent de manière dynamique la valeur d'un attribut (p. ex., titre, prix, etc.) entre des accolades : {{ ... }}

Liquid


<h1>{{ product.title }}</h1>
<p>{{ product.description }}</p>

Retour au HTML


<h1>Planche à fromage en bois, déco coq</h1>
<p>Cette planche à fromage en grain de bois avec décoration de coq est magnifique.
Elle est idéale pour les collectionneurs d'objets à l'effigie de coqs ou pour ceux cherchant à ajouter une touche de style country ou rustique à leur décor.</p>

Exercice n° 6c : Afficher l'image de premier plan, s'il y en a une

Nous pouvons utiliser les balises Control Flow pour vérifier s'il y a une image de premier plan.

Les balises Liquid sont utilisées pour programmer des éléments logiques. Les balises ressemblent à ceci : {% ... %}

Pseudocode


{% if collection_image_exists %}
    Afficher l'image de premier plan
{% endif %}

Truc de pro : le pseudocode sert à verbaliser les étapes d'un problème avec des mots simples. Ce n'est pas du vrai code : les humains peuvent le lire et la syntaxe n'a aucune importance. Il s'agit d'un brouillon que l'on peut modifier (et on peut écrire en français)!

Expressions conditionnelles (if)

Une expression conditionnelle ne fait une action que si une condition est vraie.


{% if une condition est vraie %}
    faire ceci...
{% endif %}

Elle peut faire différentes actions si des conditions différentes sont vraies.


{% if la condition primaire est vraie %}
    faire ceci
{% elsif la condition secondaire est vraie %}
    faire cela
{% else %}
    if aucune condition n'est vraie, faire ceci.
{% endif %}

Afficher un code de réduction

Comment afficher un code de réduction sur une page de produit si le produit fait partie d'une collection précise (dans ce cas-ci, la collection Household)?

  1. Créez un code de réduction qui ne fonctionnera que pour les produits de la collection Household.
  2. Modifiez product-template.liquid pour afficher le code dans une expression conditionnelle.

Créer un code de réduction

  • Code : home4holidays
  • Options : réduction de 20 %
  • S'applique aux : produits de la collection Household
  • Facultatif : indiquez une date de début et de fin si vous souhaitez que le code expire après les fêtes, par exemple.

Balises d'itération

Maintenant, nous souhaitons afficher ce code de réduction sur tous les produits qui font partie de cette collection. Comme un produit peut faire partie de plusieurs collections, vérifions d'abord quel est le nom de la collection à laquelle ce produit appartient.

Nous pouvons utiliser une balise d'itération pour consulter les collections en boucle :

Pseudocode


{% for each collection in collections %}

    {% if titre est 'Household' %}
      Afficher le code de réduction.
    {% endif %}

{% endfor %}

À la découverte de la boucle « for »

Une boucle est la répétition d'une instruction jusqu'à ce qu'une tâche soit complètement exécutée. Pensez à un évier rempli de verres à nettoyer : vous répéterez les mêmes actions pour chaque verre jusqu'à ce qu'il n'y ait plus de verres dans l'évier. En fait, vous faites une boucle!

À la découverte de la boucle « for »

Créons une boucle « for » basée sur notre pseudocode.

Quelle propriété de produit pouvons-nous utiliser pour obtenir des informations sur les collections des produits?

Réponse : {{ product.collections }}

À la découverte de la boucle « for »

Quelle propriété de collection le titre indique-t-il?

Réponse : {{ collection.title }}

Quel opérateur vérifie si deux éléments sont égaux?

Réponse : ==


{% for collection in product.collections %}
  {% if collection.title == 'Household' %}
    Afficher le code de réduction.
  {% endif %}
{% endfor %}

Voyons voir s'il est possible de déterminer l'emplacement du code des collections de la page d'accueil!

Ouvrez la section appelée collection.liquid et trouvez l'emplacement du code qui affiche les produits.

Réponse : cacher le prix (hide price)

    {% unless product.template_suffix == 'info-only' %}

Exercice n° 10d : Modifier un extrait de code

Réponse : message « sustainable » (durable)

{% if product.tags contains 'sustainable' %}
  <p>Produit durable</p>

Bogus Gateway

Simuler le processus de commande

Bogus Gateway

Vous pouvez simuler une transaction sur votre boutique. Cela est utile si vous souhaitez :

  • Personnaliser vos courriels de confirmation
  • Essayer des modifications de panier
  • Voir la page de statut de la commande

La meilleure façon d'y parvenir dans une boutique de développement est d'activer Bogus Gateway.

Votre boutique de développement vous permet de faire 10 « vraies » transactions. Toutefois, Bogus Gateway vous permet de faire des transactions illimitées.

Remarque : Bogus Gateway n'est disponible que pour les boutiques de développement créées avec un compte partenaire.

Lorsque vous achetez un produit de votre boutique et annulez la transaction ou faites un remboursement, il s'agit d'une « vraie » transaction. Si vous achetez quelque chose avec l'option « Cash on delivery », il s'agit aussi d'une « vraie » transaction.
